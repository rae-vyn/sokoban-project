# sokoban-project
A sokoban game project for CSC141

# Art (should be a constant size, like 32x32)
- A character
- Boxes
- Something to mark objective spots (like a star, or a circle or something)
- A wall texture
# Sound Effects
- Box push sound effect
	- A separate effect for pushing a box that can't be moved (ex. against a wall or another box)
- Sound effect for placing a block on an objective spot
- level complete sound
# Levels
- At least 10 levels
